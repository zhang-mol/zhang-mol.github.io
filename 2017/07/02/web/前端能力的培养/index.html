前端能力的培养

发表于 2017-07-02 | 分类于 Web | 阅读次数
| 574 (字) | 2 (分)

本人新手一个，万万不敢谈前端能力的培养，此博文借助文档-前端能力的培养-来总结个人所需，莫名的想到了《演员的自我修养》。如与您有不同习惯，不同意见，敬请莫怪。重申：本博文只属个人所需，以此来提高自身。

打造个人工具箱

工欲善其事，必先利其器。每个开发者都需要有属于个人的瑞士军刀，在工作时会为你省下很多宝贵的时间。

高效的开发环境

高效的开发环境最主要的目的就是高效率。

  • 适合的编辑器 (vscode)
  • 代码版本控制系统 (git svn)
  • 代码规格标准 (eslint)
  • 打包工具 (webpack)
    笔记系列
    一个人的大脑就像电脑的内存盘，记录的东西有限，因此你需要另外的‘存储盘’，so必须养成一个好的习惯–做笔记；不管是用markdown还是记录在博客中，至少拥有一个。
  • 信息采集器（markdown）
  • 个人博客 (molb.me)

构建个人技术体系

学如逆水行舟，不进则退。前端的技术更新太快，学习能促使你不断的进步。

  • 前端的成长可以分为两部分，一部分是‘能力’，一部分是‘知识’。能力>知识，能力包括编程能力、架构能力、工程能力。
  • 训练与积累：通过大量的训练，巩固所学所得。
  • 善于总结：去学习一个新的框架，总结能够帮助你清晰的了解整个流程。
  • 最后建议经营一款自己的产品，可以是一款APP、一个网站或者一个开源软件。成为全栈工程师的捷径就是打造一款自己的产品。

    重视沟通能力的培养

    开发往往是一个团队的工作，而善于表达的人能够在第一时间反馈精准的信息。沟通能力不管是对工作还是为人处世都是很重要的。而这要做到：
  • 自信，相信自己，才能让别人信任你
  • 随和，平易近人，才能和别人相处
  • 幽默，风趣幽默，能让你获得更多目光
  • 适度，掌握度，能走的更远

Git简介

Git是什么？Git是分布式版本控制系统。（什么是分布式版本控制系统，度娘会告诉你的）
学习Git是为了什么？很简单，是为了更好的拿高薪，为了更好的装X，为了提升本身的技能，毕竟代码的世界发展太快。

Linux上安装

查看系统是否安装Git

如果使用的是Debian或Ubuntu Linux，可通过一条sudo apt-get install git就可以直接完成Git的安装，非常简单。其他版本Linux，可去官网下载源码安装。

Mac上安装

Mac安装git有两种方法：
一是安装homebrew，然后通过homebrew安装Git，具体方法参考homebrew官方文档：http://brew.sh/。
第二种方法更简单，也是推荐的方法，就是直接从AppStore安装Xcode，Xcode集成了Git，不过默认没有安装，你需要运行Xcode，选择菜单“Xcode”->“Preferences”，在弹出窗口中找到“Downloads”，选择“Command Line Tools”，点“Install”就可以完成安装了。

windows上安装

在Windows上使用Git，可以从Git官网直接下载安装程序，（网速慢的同学请移步国内镜像），然后按默认选项安装即可。
安装完成后，在开始菜单里找到“Git”->“Git Bash”，蹦出一个类似命令行窗口的东西，就说明Git安装成功！

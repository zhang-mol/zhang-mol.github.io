在本地手动删除，使用添加时的记录还在。

  • 删除添加到暂存区的文件

    1. 只删除暂存区的文件

  • 删除提交到本地仓库的文件
    若是不仅添加到了暂存区，而且commit到本地仓库，这个时候就不能使用git rm了，需要使用git reset命令。
    一旦提交到本地仓库，此时工作区、暂存区，还是仓库，这三者的内容都是一样的。所以需要采用撤销操作。 git reset有三个选项，–hard、–mixed、–soft。详细请看Git删除暂存区或版本库里的文件
